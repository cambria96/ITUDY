# 프로젝트 소개 
---
기존의 스터디 관련 서비스는 시간, 장소, 관심사 등 개인적인 제약들 때문에 사용자에게 적합한 스터디를 구하는데 어려움있다. 따라서 이를 해결하기 위해 IT관련 공부를 희망하는 사람들을 대상으로 조건에 맞는 스터디 그룹을 매칭해주는 서비스를 제공하는 ITUDY(IT+STUDY)를 기획하였다.
